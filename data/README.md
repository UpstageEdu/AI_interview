# Amazon Food Reviews 데이터셋

## 데이터셋 개요

본 데이터셋은 **Amazon**에서 10년 이상 수집된 약 50만 개의 식품 리뷰 데이터입니다. 각 리뷰는 평점, 요약, 본문 등 다양한 정보를 포함하고 있어 텍스트 분석 및 요약 모델 학습에 유용합니다.

---

## 사용하는 데이터 필드

-   **Summary**: 리뷰의 제목 역할을 하는 한 줄 요약
-   **Text**: 리뷰의 전체 내용이 담긴 원본 텍스트
